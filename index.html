Pitfalls & Gotchas

"L'agent est une extension de votre cerveau, pas un remplacement. Voici où il peut échouer."

01

Hallucinations Silencieuses

"L'IA peut inventer une bibliothèque qui n'existe pas ou un flag imaginaire avec une certitude absolue."

FIX: Toujours valider via le Pilier 'CHECK' (LSP / Tests).
02

Saturation du Contexte

"Trop de fichiers dans le chat = dilution de l'attention de l'IA. Elle commence à oublier les instructions du début."

FIX: Utilisez /drop pour garder un contexte chirurgical.
03

Fuite de Secrets (.env)

"Envoyer vos clés API ou secrets de prod aux serveurs de l'IA est un risque majeur de conformité."

FIX: Configurez un .aiderignore ou .geminiignore strict.
04

Dette Technique "Vibe"

"Générer du code qui 'marche' mais qui est illisible ou non-maintenable sur le long terme."

FIX: Imposer des conventions via un fichier CONVENTIONS.md.
