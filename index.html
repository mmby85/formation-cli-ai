Pitfalls &
Gotchas

"L'agent est une extension de votre cerveau, pas un remplacement."

01

Hallucinations Silencieuses

"L'IA peut inventer une bibliothèque inexistante ou un flag imaginaire avec une certitude absolue."

FIX: Validez toujours avec vos tests + une recherche dans la doc officielle.
02

Saturation du Contexte

"Trop de fichiers = dilution de l'attention. L'IA oublie les instructions du début de session."

FIX: Utilisez /drop dans Aider ou /context clear dans OpenCode.
03

Fuite de Secrets (.env)

"Envoyer vos clés API ou secrets de prod aux serveurs cloud de l'IA = risque de conformité majeur."

FIX: Configurez .aiderignore et .gitignore. Utilisez Ollama en local pour le code sensible.
04

Dette Technique "Vibe"

"Générer du code qui 'marche' mais qui est illisible ou non-maintenable à long terme."

FIX: Imposez des conventions via un CONVENTIONS.md + lint automatique.
